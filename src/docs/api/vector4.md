# Vector4

**Namespace:** Microsoft.Xna.Framework

## Summary

`Vector4` represents a four-dimensional vector with X, Y, Z, and W components. Often used for colors and homogeneous coordinates.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| X | float | X component |
| Y | float | Y component |
| Z | float | Z component |
| W | float | W component |

## Code Example

```csharp
Vector4 color = new Vector4(1.0f, 0.5f, 0.0f, 1.0f); // RGBA
```
